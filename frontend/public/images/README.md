# Menu Images Folder

Place your food images here and reference them in the menu using `/images/filename.jpg`

## Usage Example:
- Save your image as: `pizza.jpg`
- In the menu form, use: `/images/pizza.jpg`

## Supported formats:
- .jpg / .jpeg
- .png
- .gif
- .webp

## Tips:
- Use descriptive filenames (e.g., `chicken-burger.jpg`, `chocolate-cake.png`)
- Keep file sizes reasonable (< 1MB recommended)
- Use lowercase filenames with hyphens for consistency
